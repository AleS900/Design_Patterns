# Ejercicio 14 - Inscripción A Cursos

Se debe realizar la *inscripción* de los estudiantes
a los cursos existentes utilizando la opción de
**inscribirse**.</br>

- Los cursos son:
```
- Matemáticas
- Lenguaje
- Historia
- Inglés
```
### 5 estudiantes deben inscribirse a alguna materia.
